## API

| Name  | Description                   | Type                                                                     | Default     |
| ----- | ----------------------------- | ------------------------------------------------------------------------ | ----------- |
| color | Label color                   | `'default' \| 'primary' \| 'success' \| 'warning' \| 'danger' \| string` | `'default'` |
| fill  | Filled mode                   | `'solid' \| 'outline'`                                                   | `'solid'`   |
| round | Whether it is rounded corners | `boolean`                                                                | `false`     |
