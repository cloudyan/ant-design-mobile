## API

| Name      | Description                 | Type                           | Default |
| --------- | --------------------------- | ------------------------------ | ------- |
| content   | The text content            | `string`                       | -       |
| direction | Position omitted            | `'start' \| 'end' \| 'middle'` | `'end'` |
| rows      | The number to display lines | `number`                       | `1`     |
