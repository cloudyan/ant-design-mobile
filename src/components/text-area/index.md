# API

| 属性         | 说明                 | 类型                    | 默认值 |
| ------------ | -------------------- | ----------------------- | ------ |
| value        | 输入值               | string                  | -      |
| defaultValue | 默认值               | string                  | -      |
| placeholder  | 提示文本             | string                  | -      |
| rows         | 行数                 | number                  | 2      |
| onChange     | 文本域内容变化时触发 | (value: string) => void | -      |

其他属性和原生 `textarea` 一致
