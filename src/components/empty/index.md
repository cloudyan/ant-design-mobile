# API

| 属性        | 说明                                       | 类型          | 默认值              |
| ----------- | ------------------------------------------ | ------------- | ------------------- |
| imageStyle  | 图片样式                                   | CSSProperties | -                   |
| description | 图片下方的描述文字                         | ReactNode     | -                   |
| image       | 设置图片，为 string 时表示自定义图片地址。 | ReactNode     | Empty.DEFAULT_IMAGE |
