## API

| 成员        | 说明           | 类型      | 默认值       |
|------------|----------------|--------------------|--------------|
| children   | 导航内容      | any |    无  |
| mode   | 导航模式   | string |  'dark' enum{'dark', 'light'} |
| iconName   | 左边icon name   | string |  'left' |
| leftContent   | 导航左边内容      | any |    无  |
| rightContent   | 导航右边内容      | any |    无  |
| onLeftClick   | 导航左边点击回调      | function |    无  |
