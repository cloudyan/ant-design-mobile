## 如何使用

上下留白,设计上通常需要一段特定高度的空白区块

## API

| 成员        | 说明           | 类型      |  可选值         | 默认值       |
|------------|----------------|--------------------|--------------|
| size       |  上下留白的间距   | string | `xs`、`sm`、`md`、`lg`、`xl` | `md`  |
