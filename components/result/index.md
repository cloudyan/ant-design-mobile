### 定义／Definition

用于系统繁忙，网络异常，返回空结果等异常页面，结果页的插画可以自定义。

## API

### Result
| 成员        | 说明           | 类型               | 默认值       |
|------------|----------------|--------------------|--------------|
| imgUrl    | 插图URL        | string |    |
| title    |    title文案     | String/HTML |    |
| message    |    message文案     | String/HTML |    |
| buttonText    |    按钮文案     | string |    |
| buttonType    |    请参考button的配置     | string |    |
| buttonClick    |    按钮回调函数     | Func |    ||
