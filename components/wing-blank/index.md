## 如何使用

设计上,通常dom两边要设置下特定宽度边距

## API

| 成员        | 说明           | 类型      |      可选值      | 默认值       |
|------------|----------------|--------------------|--------------|
| size    | 两翼留白的间距  | string | `sm`、`md`、`lg` |  `lg`  |
